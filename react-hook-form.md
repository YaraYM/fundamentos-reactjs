# React Hook Form

- Uma das principais bibliotecas de formulários para React
- Adiciona funcionalidade ao nosso formulário
- Possível trabalhar com formulários tanto de forma controlled quanto uncontrolled - é possível ter performace sem abrir mão da flexibilidade, da fluidez, da interatividade dos campos do nosso formulário
- Por padrão, essa biblioteca não traz nada de validação - prefere se manter uma biblitoeca mais 'enxuta', com menos funcionalidades, e utilizar outras bibliotecas feitas para validação integradas a ele (exemplos: yup, joi, zod)
