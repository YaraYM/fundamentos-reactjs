# Bundlers & Compilers

- Nem sempre os browsers suportam a versão mais atualizada do JS que estamos utilizando.


## Bundlers

- Ferramenta capaz de pegar todos os arquivos JS da aplicação e transformá-los em um só.
- Ex.: Webpack.


## Compilers

- Compiladores de código.
- Ferramentas que convertem o código de um formato para outro, de uma versão para outra.
- Ex.: Babel.
