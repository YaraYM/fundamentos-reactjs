# Programação imperativa

- É dito para o software o que deve ser feito

## Receita de bolo

1. Ligar o forno a 180°
2. Abrir a porta do forno
3. Colocar a massa numa assadeira
4. Colocar a forma com a massa dentro do forno

# Programação declarativa

- Ao invés de dizer como se espera que o resultado final seja atingido, passo-a-passo, declaramos apenas o resultado esperado
- Quais as condições para eu ter o resultado final

## Receita de bolo

1. O forno precisa estar a 180°
2. Quando o forno estiver quente, eu posso colocar a massa para assar
3. Quando a massa estiver pronta, eu posso retirá-la do forno
